# OOP-Coursework-Project
This repository will contain my Java code for mt Object-Oriented-Programming coursework project. It is a image managing system which can add and view different images.
The system is able to display the details of an image e.g. name, description, genre, date, thumbnail and display the image itself using a supplied console class.
Part01 is a text-only verision which uses Eclipse's (or similar IDEs/programming environment) console to display text and take in user input. 
Part 02 uses Queen's University Belfast own console which has the ability to clear the console of text, and display images and text, and take in user input. It builds upon the code in Part01 by modifying it to support the console and include more features related to the specific ability of the console.
